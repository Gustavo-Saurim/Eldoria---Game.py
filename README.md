# 🎮 Eldoria: Fragmentos do Abismo

## 📖 Descrição
Eldoria é um RPG de texto imersivo atualmente em desenvolvimento. O jogo já conta com seu primeiro capítulo completo, introduzindo os jogadores a um universo rico em narrativa e escolhas. Novas funcionalidades, mecânicas aprimoradas e inimigos desafiadores estão em constante desenvolvimento, prometendo expandir ainda mais a experiência.
